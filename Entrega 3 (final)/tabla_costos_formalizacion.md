## EL SALTO TRIBUTARIO: COMPARACIÓN DE REGÍMENES TRIBUTARIOS (2021)

**Caso:** Empresa que pasa de S/8,000 a S/8,001 de ingresos mensuales | Margen comercial: 30.97%*

| **Concepto** | **RUS** | **RER** | **Régimen MYPE** |
|--------------|---------|---------|------------------|
| **Ingresos mensuales** | S/8,000 | S/8,001 | S/8,001 |
| **Ingresos anuales** | S/96,000 | S/96,012 | S/96,012 |
| **Utilidad bruta anual (30.97%)** | S/29,731 | S/29,735 | S/29,735 |
| | | | |
| **Costos tributarios anuales:** | | | |
| - Pago único RUS | S/600 | - | - |
| - Impuesto a la Renta RER (1.5% ingreso mensual) | - | S/1,440.18 | - |
| - Impuesto a la Renta MYPE (proxy: 10% utilidad bruta) | - | - | S/2,974 |
| - Arbitrios municipales | S/500 | S/500 | S/500 |
| **Total costos tributarios** | **S/1,100** | **S/1,940.18** | **S/3,474** |
| | | | |
| **% costos sobre ingresos** | **1.15%** | **2.02%** | **3.62%** |
| **% costos sobre utilidad bruta** | **3.7%** | **6.5%** | **11.7%** |
| **Diferencia vs RUS** | Base | **+S/840.18** | **+S/2,374** |

*Margen comercial promedio del V Censo Económico Nacional 2022 (1,282,008 microempresas). Distribución por régimen: RUS 36.3%, RER 6.1%, RMT 6.4%.

### Análisis del salto tributario:
1. **Cambio de régimen forzoso:** No hay opción de permanecer en RUS si los ingresos superan S/8,000
2. **Escalonamiento de costos:** 
   - RUS: S/1,100 (3.7% de utilidad bruta)
   - RER: S/1,940 (6.5% de utilidad bruta) - aumento del 76.4% 
   - MYPE: S/3,474 (11.7% de utilidad bruta) - aumento del 215.8% (Relativamente inflado, dado que el 10% se calcula sobre utilidad neta, no bruta)
3. **RER como opción intermedia:** Aunque menos costoso que MYPE, el RER tiene múltiples restricciones de actividades
4. **Impacto desproporcionado:** Por S/12 adicionales de ingresos anuales (0.0125%), los costos aumentan entre S/840 (RER) y S/2,374 (MYPE)
5. **Desincentivo al crecimiento:** El salto al Régimen MYPE triplica los costos tributarios

**Nota sobre márgenes:** Según el censo, las microempresas informales tienen margen comercial de 34.01% vs 29.31% de las formales, sugiriendo un costo oculto adicional de 4.7 puntos porcentuales por la formalidad. Dado que el margen comercial está directamente relacionado con la utilidad bruta, probablemente dicha diferencia se deba a la distorsión que genera la aplicación del 18% de IGV.

## ğŸ”§ Common Issues & Solutions

### Issue 1: "pandoc: command not found"
**Solution:** Install pandoc
```bash
brew install pandoc
```

### Issue 2: "No LaTeX engine found"
**Solution:** Install BasicTeX or MacTeX
```bash
brew install --cask basictex
sudo tlmgr update --self
```

### Issue 3: Missing LaTeX packages during conversion
**Solution:** Install specific package
```bash
sudo tlmgr install <package-name>
# Example:
sudo tlmgr install collection-fontsrecommended
```

### Issue 4: Equations not rendering in PDF
**Solution:** Check your LaTeX syntax in the markdown files
- Use `$$...$$` for display equations
- Use `$...$` for inline math
- Use `\frac{a}{b}` for fractions
- Use `\times` for multiplication
- Use `\beta_1` for subscripts

### Issue 5: Spanish characters not showing (Ã¡, Ã©, Ã­, Ã³, Ãº, Ã±)
**Solution:** Already handled! The script uses `xelatex` engine which supports UTF-8
